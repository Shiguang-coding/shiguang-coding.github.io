Dockerз®Җд»Ӣ

DockerжҳҜеҹәдәҺGoиҜӯиЁҖе®һзҺ°зҡ„дә‘ејҖжәҗйЎ№зӣ®гҖӮ

Dockerзҡ„дё»иҰҒзӣ®ж ҮжҳҜпјҡBuild, Ship and Run Any App, AnywhereпјҢд№ҹе°ұжҳҜйҖҡиҝҮеҜ№еә”з”Ёз»„д»¶зҡ„е°ҒиЈ…гҖҒеҲҶеҸ‘гҖҒйғЁзҪІгҖҒиҝҗиЎҢзӯүз”ҹе‘Ҫе‘Ёжңҹзҡ„з®ЎзҗҶпјҢдҪҝз”ЁжҲ·зҡ„APPеҸҠе…¶иҝҗиЎҢзҺҜеўғиғҪеҒҡеҲ°дёҖж¬Ўй•ңеғҸ,еӨ„еӨ„иҝҗиЎҢгҖӮ

дј з»ҹиҷҡжӢҹжңәе’Ңе®№еҷЁ

дј з»ҹиҷҡжӢҹжңәпјҲvirtual machineпјүпјҡ

дј з»ҹиҷҡжӢҹжңәжҠҖжңҜеҹәдәҺе®үиЈ…еңЁдё»ж“ҚдҪңзі»з»ҹдёҠзҡ„иҷҡжӢҹжңәз®ЎзҗҶзі»з»ҹпјҲеҰӮVirtualBoxгҖҒVMwareзӯүпјүпјҢеҲӣе»әиҷҡжӢҹжңәпјҲиҷҡжӢҹеҮәеҗ„з§ҚзЎ¬д»¶пјүпјҢеңЁиҷҡжӢҹжңәдёҠе®үиЈ…д»Һж“ҚдҪңзі»з»ҹпјҢеңЁд»Һж“ҚдҪңзі»з»ҹдёӯе®үиЈ…йғЁзҪІеҗ„з§Қеә”з”ЁгҖӮ

зјәзӮ№пјҡиө„жәҗеҚ з”ЁеӨҡгҖҒеҶ—дҪҷжӯҘйӘӨеӨҡгҖҒеҗҜеҠЁж…ў

Linuxе®№еҷЁпјҲLinux ContainerпјҢз®Җз§°LXCпјүпјҡ

Linuxе®№еҷЁжҳҜдёҺзі»з»ҹе…¶д»–йғЁеҲҶеҲҶйҡ”ејҖзҡ„дёҖзі»еҲ—иҝӣзЁӢпјҢд»ҺеҸҰдёҖдёӘй•ңеғҸиҝҗиЎҢпјҢе№¶з”ұиҜҘй•ңеғҸжҸҗдҫӣж”ҜжҢҒиҝӣзЁӢжүҖйңҖзҡ„е…ЁйғЁж–Үд»¶гҖӮе®№еҷЁжҸҗдҫӣзҡ„й•ңеғҸеҢ…еҗ«дәҶеә”з”Ёзҡ„жүҖжңүдҫқиө–йЎ№пјҢеӣ иҖҢеңЁд»ҺејҖеҸ‘еҲ°жөӢиҜ•еҶҚеҲ°з”ҹдә§зҡ„ж•ҙдёӘиҝҮзЁӢдёӯпјҢе®ғйғҪе…·жңүеҸҜз§»жӨҚжҖ§е’ҢдёҖиҮҙжҖ§гҖӮ

Linuxе®№еҷЁдёҚжҳҜжЁЎжӢҹдёҖдёӘе®Ңж•ҙзҡ„ж“ҚдҪңзі»з»ҹпјҢиҖҢжҳҜеҜ№иҝӣзЁӢиҝӣиЎҢйҡ”зҰ»гҖӮжңүдәҶе®№еҷЁпјҢе°ұеҸҜд»Ҙе°ҶиҪҜд»¶иҝҗиЎҢжүҖйңҖзҡ„жүҖжңүиө„жәҗжү“еҢ…еҲ°дёҖдёӘйҡ”зҰ»зҡ„е®№еҷЁдёӯгҖӮе®№еҷЁдёҺиҷҡжӢҹжңәдёҚеҗҢпјҢдёҚйңҖиҰҒжҚҶз»‘дёҖж•ҙеҘ—ж“ҚдҪңзі»з»ҹпјҢеҸӘйңҖиҰҒиҪҜд»¶е·ҘдҪңжүҖйңҖзҡ„еә“иө„жәҗе’Ңи®ҫзҪ®гҖӮзі»з»ҹеӣ жӯӨиҖҢеҸҳеҫ—й«ҳж•ҲиҪ»йҮҸе№¶дҝқиҜҒйғЁзҪІеңЁд»»дҪ•зҺҜеўғдёӯзҡ„иҪҜд»¶йғҪиғҪе§Ӣз»ҲеҰӮдёҖзҡ„иҝҗиЎҢгҖӮ

еҜ№жҜ”пјҡ

зү№жҖ§ е®№еҷЁ иҷҡжӢҹжңә
еҗҜеҠЁ з§’зә§ еҲҶй’ҹзә§
еӨ§е°Ҹ дёҖиҲ¬дёәMb дёҖиҲ¬дёәGb

DockerиҝҗиЎҢйҖҹеәҰеҝ«зҡ„еҺҹеӣ 

DockerжңүжҜ”иҷҡжӢҹжңәжӣҙе°‘зҡ„жҠҪиұЎеұӮпјҡ

з”ұдәҺDockerдёҚйңҖиҰҒHypervisorпјҲиҷҡжӢҹжңәпјүе®һзҺ°зЎ¬д»¶иө„жәҗиҷҡжӢҹеҢ–пјҢиҝҗиЎҢеңЁDockerе®№еҷЁдёҠзҡ„зЁӢеәҸзӣҙжҺҘдҪҝз”Ёзҡ„йғҪжҳҜе®һйҷ…зү©зҗҶжңәзҡ„зЎ¬д»¶иө„жәҗпјҢеӣ жӯӨеңЁCPUгҖҒеҶ…еӯҳеҲ©з”ЁзҺҮдёҠdockerжңүжҳҺжҳҫдјҳеҠҝгҖӮ

DockerеҲ©з”Ёзҡ„жҳҜе®ҝдё»жңәзҡ„еҶ…ж ёпјҢиҖҢдёҚйңҖиҰҒеҠ иҪҪж“ҚдҪңзі»з»ҹOSеҶ…ж ёпјҡ

еҪ“ж–°е»әдёҖдёӘе®№еҷЁж—¶пјҢDockerдёҚйңҖиҰҒе’ҢиҷҡжӢҹжңәдёҖж ·йҮҚж–°еҠ иҪҪдёҖдёӘж“ҚдҪңзі»з»ҹеҶ…ж ёгҖӮиҝӣиҖҢйҒҝе…Қеј•еҜ»гҖҒеҠ иҪҪж“ҚдҪңзі»з»ҹеҶ…ж ёиҝ”еӣһзӯүжҜ”иҫғиҖ—ж—¶иҖ—иө„жәҗзҡ„иҝҮзЁӢгҖӮеҪ“ж–°е»әдёҖдёӘиҷҡжӢҹжңәж—¶пјҢиҷҡжӢҹжңәиҪҜд»¶йңҖиҰҒеҠ иҪҪOSпјҢиҝ”еӣһж–°е»әиҝҮзЁӢжҳҜеҲҶй’ҹзә§еҲ«зҡ„гҖӮиҖҢDockerз”ұдәҺзӣҙжҺҘеҲ©з”Ёе®ҝдё»жңәзҡ„ж“ҚдҪңзі»з»ҹпјҢеҲҷзңҒз•ҘдәҶиҝ”еӣһиҝҮзЁӢпјҢеӣ жӯӨж–°е»әдёҖдёӘdockerе®№еҷЁеҸӘйңҖиҰҒеҮ з§’й’ҹгҖӮ

Dockerе®№еҷЁзҡ„жң¬иҙЁе°ұжҳҜдёҖдёӘиҝӣзЁӢгҖӮ

DockerиҪҜд»¶

Dockerе№¶йқһдёҖдёӘйҖҡз”Ёзҡ„е®№еҷЁе·Ҙе…·пјҢе®ғдҫқиө–дәҺе·Із»ҸеӯҳеңЁе№¶иҝҗиЎҢзҡ„LinuxеҶ…ж ёзҺҜеўғгҖӮпјҲеңЁWindowsдёҠе®үиЈ…Dockerж—¶йңҖиҰҒдҫқиө–WLSпјҢд№ҹеҚіWindowsдёӢзҡ„Linuxеӯҗзі»з»ҹпјүгҖӮ

Dockerе®һиҙЁдёҠжҳҜеңЁе·Із»ҸиҝҗиЎҢзҡ„LinuxдёӢеҲ¶йҖ дәҶдёҖдёӘйҡ”зҰ»зҡ„ж–Үд»¶зҺҜеўғпјҢеӣ жӯӨе®ғжү§иЎҢзҡ„ж•ҲзҺҮеҮ д№ҺзӯүеҗҢдәҺжүҖйғЁзҪІзҡ„Linuxдё»жңәгҖӮ

Dockerзҡ„еҹәжң¬з»„жҲҗйғЁеҲҶпјҡ

в—Ҹй•ңеғҸпјҲimageпјү

в—Ҹе®№еҷЁпјҲcontainerпјү

в—Ҹд»“еә“пјҲrepositoryпјү

Dockerй•ңеғҸ

Dockerй•ңеғҸе°ұжҳҜдёҖдёӘеҸӘиҜ»зҡ„жЁЎжқҝгҖӮй•ңеғҸеҸҜд»Ҙз”ЁжқҘеҲӣе»әDockerе®№еҷЁпјҢдёҖдёӘй•ңеғҸеҸҜд»ҘеҲӣе»әеӨҡдёӘе®№еҷЁгҖӮ

Dockerе®№еҷЁ

DockerеҲ©з”Ёе®№еҷЁзӢ¬з«ӢиҝҗиЎҢзҡ„дёҖдёӘжҲ–дёҖз»„еә”з”ЁпјҢеә”з”ЁзЁӢеәҸжҲ–жңҚеҠЎиҝҗиЎҢеңЁе®№еҷЁйҮҢйқўпјҢе®№еҷЁе°ұзұ»дјјдәҺдёҖдёӘиҷҡжӢҹеҢ–зҡ„иҝҗиЎҢзҺҜеўғпјҢе®№еҷЁжҳҜз”Ёй•ңеғҸеҲӣе»әзҡ„иҝҗиЎҢе®һдҫӢгҖӮ

Dockerд»“еә“

Dockerд»“еә“жҳҜйӣҶдёӯеӯҳж”ҫй•ңеғҸж–Үд»¶зҡ„еңәжүҖгҖӮ

д»“еә“еҲҶдёәе…¬ејҖд»“еә“е’Ңз§Ғжңүд»“еә“дёӨз§ҚгҖӮ

жңҖеӨ§зҡ„е…¬ејҖд»“еә“жҳҜDockerе®ҳж–№зҡ„Docker Hubпјҡhttps://hub.docker.com/

Dockerжһ¶жһ„

DockerжҳҜдёҖдёӘ C/SпјҲClient-Serverпјү з»“жһ„зҡ„зі»з»ҹпјҢеҗҺз«ҜжҳҜдёҖдёӘжқҫиҖҰеҗҲжһ¶жһ„пјҢдј—еӨҡжЁЎеқ—еҗ„еҸёе…¶иҒҢгҖӮ

Dockerе®ҲжҠӨиҝӣзЁӢиҝҗиЎҢеңЁдё»жңәдёҠпјҢз„¶еҗҺйҖҡиҝҮSocketиҝһжҺҘд»Һе®ўжҲ·з«Ҝи®ҝй—®пјҢе®ҲжҠӨиҝӣзЁӢд»Һе®№еҷЁжҺҘж”¶е‘Ҫд»Өе№¶з®ЎзҗҶиҝҗиЎҢеңЁдё»жңәдёҠзҡ„е®№еҷЁгҖӮ

DockerиҝҗиЎҢзҡ„еҹәжң¬жөҒзЁӢдёәпјҡ

1з”ЁжҲ·жҳҜдҪҝз”ЁDocker Client дёҺ Docker Daemon е»әз«ӢйҖҡдҝЎпјҢе№¶еҸ‘йҖҒиҜ·жұӮз»ҷеҗҺиҖ…

2Docker Daemon дҪңдёә Docker жһ¶жһ„зҡ„дё»дҪ“йғЁеҲҶпјҢйҰ–е…ҲжҸҗдҫӣ Docker Server зҡ„еҠҹиғҪдҪҝе…¶еҸҜд»ҘжҺҘж”¶ Docker Client зҡ„иҜ·жұӮ

3Docker Engine жү§иЎҢ Docker еҶ…йғЁзҡ„дёҖзі»еҲ—е·ҘдҪңпјҢжҜҸдёҖйЎ№е·ҘдҪңйғҪжҳҜд»ҘдёҖдёӘ Job зҡ„еҪўејҸеӯҳеңЁ

4Job зҡ„иҝҗиЎҢиҝҮзЁӢдёӯпјҢеҪ“йңҖиҰҒе®№еҷЁй•ңеғҸж—¶пјҢеҲҷд»Һ Docker Registry дёӯдёӢиҪҪй•ңеғҸпјҢе№¶йҖҡиҝҮй•ңеғҸз®ЎзҗҶй©ұеҠЁ Graph Driver е°ҶдёӢиҪҪй•ңеғҸд»Ҙ Graph зҡ„еҪўејҸеӯҳеӮЁ

5еҪ“йңҖиҰҒдёә Docker еҲӣе»әзҪ‘з»ңзҺҜеўғж—¶пјҢйҖҡиҝҮзҪ‘з»ңз®ЎзҗҶй©ұеҠЁ Network driver еҲӣе»әе№¶й…ҚзҪ® Docker е®№еҷЁзҪ‘з»ңзҺҜеўғ

6еҪ“йңҖиҰҒйҷҗеҲ¶ Docker е®№еҷЁиҝҗиЎҢиө„жәҗжҲ–жү§иЎҢз”ЁжҲ·жҢҮд»Өзӯүж“ҚдҪңж—¶пјҢеҲҷйҖҡиҝҮ Exec driver жқҘе®ҢжҲҗ

7Libcontainer жҳҜдёҖйЎ№зӢ¬з«Ӣзҡ„е®№еҷЁз®ЎзҗҶеҢ…пјҢNetwork driver д»ҘеҸҠ Exec driver йғҪжҳҜйҖҡиҝҮ Libcontainer жқҘе®һзҺ°е…·дҪ“еҜ№е®№еҷЁиҝӣиЎҢзҡ„ж“ҚдҪң
