全球首个破 1000 tokens/s 的 AI 编程助手！小米 MiMo Code 开源发布，内置模型限时免费

時光2026-06-132026-06-18

小米MiMo首次进入Coding Agent领域，Agent能力全球第3，逼近Claude Opus 4.6

🚀 快速体验入口

  • 官方网站: https://mimo.xiaomi.com/mimocode
  • 在线体验: https://aistudio.xiaomimimo.com/
  • API接入: https://platform.xiaomimimo.com/
  • MiMo Code安装:
    • Mac/Linux: curl -fsSL https://mimo.xiaomi.com/install | bash
    • Windows: npm install -g @mimo-ai/cli
    • 启动命令: mimo

🔥 核心亮点（快速了解）

  • ✅ 完全免费体验：内置MiMo-V2.5多模态模型，无需注册即可使用，限时免费通道已开启[1]
  • ✅ 全球最快推理速度：2026年6月8日上线UltraSpeed模式，全球首个突破1000 tokens/s的万亿参数模型[2]
  • ✅ 性能逼近Claude Opus 4.6：Agent基准测试全球第3，编码能力超越Claude Sonnet 4.6[3]
  • ✅ 完全开源：MIT协议，支持接入DeepSeek、Kimi、GLM等主流模型[1]
  • ✅ 极简安装：一行命令完成安装，Mac/Linux/Windows全平台支持[1]
  • ✅ 语音控制：内置MiMo-V2.5-ASR语音识别，全程免键盘操作[1]

一、产品发布：从”会聊天”到”能干活”

2026年6月11日凌晨，小米MiMo官方正式发布并开源MiMo Code V0.1.0——一款运行在终端里的探索性AI编程助手。这标志着小米MiMo首次进入Coding Agent领域，开始构建”模型+Agent”的生态闭环[1]。

🎁 限时免费体验，无需注册

MiMo Code基于开源项目OpenCode二次开发，采用MIT协议开源发布。最吸引人的是：

  • 内置MiMo-V2.5多模态模型，提供限时免费通道
  • 无需注册即可使用，安装后立即体验
  • 同时支持接入DeepSeek、Kimi、GLM等主流模型，以及第三方Token Plan[1]

💡 开发者福利：相比Claude Code等需要付费订阅的AI编程助手，MiMo Code的免费策略极具吸引力，是体验顶级AI编程能力的绝佳机会。

安装使用极简（30秒完成）

所有设置项全面中文汉化，TUI页面右侧常驻状态看板，随时观察工作进度[1]。

二、核心创新：五大技术突破让AI”真正能干活”

1. 持久记忆系统：解决”越用越忘”难题

MiMo Code独创三重记忆机制：项目记忆、会话检查点、任务进度。通过主agent与subagent分离架构，让记录完全外包——由独立subagent自动保存状态，窗口快满时重建一份干净简报，主agent接着干而非从零开始。

实际效果：即使上百轮的长程会话，也能保持输出质量、不丢关键信息。

2. Compose模式：一键触发全流程自动化

为MiMo系列模型量身打造专属Harness系统，让模型能力与框架深度配合。用户按Tab键切换Compose模式，给出一个简单想法，就能自动完成设计、规划、编码、测试、审查的全流程，最终交付工业级成品，实现1+1>2的协同效果。

3. /dream命令：让AI”带着理解持续成长”

每7天自动触发的/dream命令，由独立Agent读取历史会话和现有记忆文件，执行合并、去重、验证路径有效性和压缩，将分散的记忆收敛为一份紧凑的当前状态，并更新全局记忆。

核心理念：MiMo Code不会每次从零开始，而是带着对项目的理解持续成长。

4. 语音全链路控制：全程免键盘操作

采用MiMo-V2.5-ASR语音识别能力，支持口头修改写错的指令，可直接发出”发送””执行”等操作命令。从输入到操控，全程不用碰键盘[1]。

5. 模型与框架深度耦合

针对MiMo模型进行专项适配与优化，让模型在代码理解、任务规划、工具调用等工程开发场景中，充分释放顶级开源模型能力。实测表现：在相同MiMo模型条件下，MiMo Code在代码任务上的表现优于Claude Code等Coding Agent框架。

三、模型能力：MiMo-V2-Pro的全球竞争力

Agent能力进入全球第一梯队

2026年3月18日，小米发布MiMo-V2-Pro——专为真实场景Agent工作负载设计的旗舰级基础模型。根据Artificial Analysis Intelligence Index（全球权威综合智能基准）[3]：

  • 全球综合智能排名第8
  • 中国大模型第2名
  • Agent基准测试全球第3（PinchBench 81.0分，逼近Claude Opus 4.6的81.5分）

万亿参数高效架构

  • 总参数量：超过1万亿
  • 激活参数：42B（约为MiMo-V2-Flash的3倍）
  • 混合注意力机制：比例从5:1提升至7:1
  • 上下文窗口：支持最长1M tokens
  • 推理速度突破：2026年6月8日上线UltraSpeed模式，全球首个在通用GPU上推理速度突破1000 tokens/s的万亿参数模型[2]

编码能力超越Claude Sonnet 4.6

内部工程师深度评估显示，MiMo-V2-Pro的编码能力超越Claude 4.6 Sonnet，体验接近Claude Opus 4.6，具备更强的系统设计和任务规划能力，代码风格更优雅，问题解决路径更高效[3]。

四、生态布局：与五大Agent框架深度集成

MiMo-V2-Pro是OpenClaw（开源社区中广受欢迎的通用Agent框架）的核心引擎，并与五大Agent开发框架达成合作[3]：

  1. OpenClaw
  2. OpenCode
  3. KiloCode
  4. Blackbox
  5. Cline

合作福利：为全球开发者提供一周免费API接入。

五、价格优势：仅为Claude Opus 4.6的1/4

MiMo-V2-Pro的API定价极具竞争力（每百万tokens）[3]：

模型 Input Output Cache Read Cache Write
MiMo-V2-Pro（≤256K） $1 $3 $0.20 $0（限时免费）
MiMo-V2-Pro（256K-1M） $2 $6 $0.40 $0（限时免费）
Claude Sonnet 4.6 $3 $15 $0.30 $3.75
Claude Opus 4.6 $5 $25 $0.50 $6.25

价格对比：相比Claude Opus 4.6，MiMo-V2-Pro价格约为其1/4-1/5，且Cache Write限时免费，进一步降低使用成本[3]。

六、战略意义：从单一模型到完整生态

补上小米AI生态关键拼图

MiMo Code的发布，标志着小米AI正从单一模型能力走向更完整的生态能力建设，构建了”模型+Agent”的生态闭环，契合AI行业从”会聊天”走向”能干活”的发展趋势。

未来行业竞争将从单一模型能力竞争转向模型、Agent与开发生态的综合竞争。

推动行业生态开放创新

MiMo Code发布当天同步开源，开发者可基于项目进行二次开发和功能扩展，有望进一步推动AI行业生态的开放和创新。

完善小米AI整体布局

结合小米此前在模型能力、推理效率、使用成本等维度的突破，MiMo Code的发布让小米AI形成了覆盖技术、应用和生态的完整布局，进展远超外界预期。

  1. 高复杂度推理

结语

小米MiMo Code的发布，不仅是一次技术产品的迭代，更是小米在AI领域战略布局的重要一步。通过构建”模型+Agent”的生态闭环，小米正在从单一的模型提供商转向全方位的AI生态建设者。

最吸引人的是：MiMo Code内置的MiMo-V2.5多模态模型 currently 提供限时免费通道，无需注册即可使用。对于想要体验顶级AI编程能力的开发者来说，这是一个不容错过的机会。

在全球AI竞赛日趋激烈的当下，小米凭借MiMo系列模型的技术积累和MiMo Code的开源策略，有望在Coding Agent领域占据一席之地，为中国AI产业的发展贡献重要力量。
