大模型 API 免费资源汇总：国内外 30+ 平台推荐

時光2026-05-312026-06-18

免费策略主要分三种：① 永久免费模型（如智谱GLM-4-Flash）、② 新用户赠送Token额度（动辄千万级）、③ 按天/按月赠送请求次数。以下是系统梳理。

强烈推荐大家用 Cherry Studio 来管理接入各个模型

一、国内平台（无需魔法，延迟低，中文能力强）

智谱AI（GLM系列）

官网地址：https://www.bigmodel.cn/invite?icode=hJQx8X%2FPL5ShGcY0IcGaIuZLO2QH3C0EBTSr%2BArzMw4%3D

  • 免费额度：新用户注册即送 2000万 Token（永久有效）

  • 免费模型：GLM-4.7-Flash、GLM-4-Flash-250414（永久），GLM-4.1V-Thinking-Flash，CogVideoX-Flash 等多个免费模型。

  • 接口地址：https://open.bigmodel.cn/api/paas/v4/

  • 亮点：代码能力国内第一梯队，OpenAI兼容格式，接入简单。是长期稳定免费方案的兜底首选。

    要注意，这个卷并非购买商品的优惠卷，而是在你调用API消耗tokens时，会自动消耗卷中的金额！
    但是！在首次使用硅基流动平台时，不会自动使用代金卷！
    我们需要在左侧余额界面先完成一次充值才能正常调用！
    可以自定义数目，输入0.01,完成一次购买后，系统会优先消耗掉这0.01元，随后消耗的tokens将会从代金卷里面扣

硅基流动（SiliconFlow）

官网地址：https://cloud.siliconflow.cn/i/h9wklc21

  • 免费额度：新用户送2000万Token（约14元），9B以下模型永久免费，实名认证送16 元全平台通用代金券
  • 速率上限：每个模型 1000 RPM（每分钟请求）
  • 免费模型：DeepSeek-R1-0528、Qwen3-8B、GLM-4-9B-Chat、DeepSeek-OCR等十余款
  • 接口地址：https://api.siliconflow.cn/v1
  • 亮点：国内服务器延迟低（常<100ms），并发容量大，适合高频调用场景

阿里云百炼（通义千问）

官网地址：https://bailian.console.aliyun.com/

  • 免费额度：每个模型 100万 Token（有效期3个月），覆盖通义千问全系、DeepSeek全系、Kimi、MiniMax、GLM等
  • 亮点：模型种类最全，每个模型独立免费额度，适合多模型对比评测

火山引擎（豆包/Doubao）

官网地址：https://console.volcengine.com/

  • 免费额度：每天200万 Token（按天重置！）+ 一次性50万Token安心体验
  • 免费模型：Doubao-lite-32k
  • 亮点：每日额度极大，适合日常高频使用

百度智能云千帆（文心一言）

官网地址：https://console.bce.baidu.com/qianfan/overview

  • 永久免费模型：ERNIE-Speed-8K（QPS=50）
  • 免费额度：ERNIE-4.5系列每模型100万Token（约3个月）
  • 亮点：ERNIE-Speed永久免费，百度品牌背书稳定可靠

月之暗面 Kimi

官网地址：https://platform.moonshot.cn/

  • 免费额度：每分钟最多3次请求，Token消耗不设每日上限；个人认证再送15元
  • 核心特性：256K 超长上下文窗口
  • 接口地址：https://platform.moonshot.cn/
  • 亮点：不限Token只限频率，处理超长文档的首选

七牛云AI推理

官网地址：https://s.qiniu.com/zuEFJv

  • 免费额度：注册即送300万token，邀请好友立返 300万 Token

  • 接口地址：https://api.qnaigc.com/v1

讯飞星辰 MaaS 平台

官网地址：https://maas.xfyun.cn/modelSquare

Qwen3.6-35B-A3B 多模态模型限时免费使用，免费至 2026年6月底（约剩不到3周）

API体验需实名认证

Qwen3.6-35B-A3B 特点：

  • 35B 总参数，MoE稀疏架构（推理时只激活 3B 参数）
  • 原生支持 262K 上下文
  • 对 Agent 编程做了专门优化，适合 Claude Code 类工作流
  • Apache 2.0 开源协议

🔧 如何使用

  1. 访问 https://maas.xfyun.cn/modelSquare
  2. 找到 Qwen3.6-35B-A3B 模型
  3. 创建应用，获取 API Key
  4. 兼容 OpenAI 格式配置到你的工具中

Free Model AI

FREE API

官网地址：https://freeapi.514179.xyz/

点击生成API即可免费获得API key，可重复生成

GitHub — chatanywhere/GPT_API_free

项目地址：GitHub — chatanywhere/GPT_API_free

这是一个长期维护的公益中转项目，提供公开共享的 API Key，每天 200 次免费调用。

  • 接口地址：
    • 国内：https://api.chatanywhere.tech
    • 国外 ：https://api.chatanywhere.org
  • 额度：每天 200 次
  • 说明：到 GitHub 项目页面查看最新的公用 Key，无需注册，拿到即用

Pollinations.AI

官网：pollinations.ai

这是目前最纯粹的公益免费站点——不需要注册，不需要 API Key，国内直连，完全开源。

  • 文本生成（OpenAI 兼容）：https://text.pollinations.ai/openai

  • 图生成：https://image.pollinations.ai/prompt/{你的描述}

  • 支持模型：DeepSeek R1 系列、OpenAI 系列、Llama 系列、Gemini 系列、Mistral 等,

    可查看模型列表：https://gen.pollinations.ai/models

  • 多模态：支持图片理解（Vision）、文生音频、语音转文字

  • 限制：基本无限制，但高并发时会排队
