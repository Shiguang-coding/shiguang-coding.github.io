文件已打开，操作无法完成解决办法

发表于2024-05-15|更新于2024-06-29|小技能
|字数总计:357|阅读时长:1分钟|阅读量:

一、问题描述

我们对文件或文件夹进行操作时，经常会遇到因文件已在其他地方打开导致操作失败的情况，而很多时候我们又不知道哪些程序打开的文件，所以遇到这个问题也很令人头疼

这里我提供两种解决方案

二、解决方法

第一种：通过资源监视器结束进程

通过快捷键 Ctrl + Shift + Esc 或者 Ctrl + Alt + Del 打开任务管理器

切换到性能侧边栏，打开资源监视器

在 关联的句柄地址栏输入要解锁的文件夹，点击右侧图标即可自动检索关联的应用程序，点击结束进程即可

第二种：通过lockhunter软件解锁

LockHunter 是一款万无一失的文件解锁器

它是一个免费工具，可以删除被您不知道的东西阻止的文件。 LockHunter 对于对抗恶意软件和其他无故阻止文件的程序非常有用。与其他类似工具不同，它会将文件删除到回收站中，因此如果错误删除，您可以恢复它们。

安装好后在被锁住的文件或文件夹上右键，点击 What's locking this file 即可查看哪些进程占用了此文件

选中占用该文件的程序，点击 Unlock It 即可进行解锁
