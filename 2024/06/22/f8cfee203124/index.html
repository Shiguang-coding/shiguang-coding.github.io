1、执行 npm link 创建链接

进入到插件所属目录以管理员身份执行 npm link 创建链接

创建的符号链接保存在全局 npm 目录中。具体来说，npm link 命令会在全局 node_modules 目录中创建一个符号链接，指向你本地包的目录。

全局 node_modules 目录的位置取决于你的操作系统和 npm 配置。通常情况下，它通常位于以下路径：

  • Linux/macOS: /usr/local/lib/node_modules
  • Windows: C:\Users\<YourUsername>\AppData\Roaming\npm\node_modules
